# movie_comment_analysis
## 利用猫眼API爬取《我不是药神》和《邪不压正》电影评论，绘制出全国各地区评论人数，同时制作热评词云，并统计打分
### 猫眼API：http://m.maoyan.com//mmdb/comments/movie/movieID(每部电影都有各自的ID，可通过猫眼电影官网查看).json?_v_=yes&
